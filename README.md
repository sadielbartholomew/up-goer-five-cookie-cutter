# Up Goer Five Cookie Cutter

## A cookie-cutter template for generating an Up Goer Five Challenge microsite of definitions and analysis 

Note that everything said here, from now on, is using the ten-hundred most used words in the spirit of the game!

### Concept

You can use this by taking the same thing again yourself - it is made as a ready-made shape to fill in with your own information. So take this and fill in the parts to make it your own.

This idea can be used to help think about the way that we talk about something and how we do it in a simple and clear way.

### Things to help you fill in parts yourself

Thing to check if you can explain a hard idea using only the ten hundred most used words by typing in a box: https://splasho.com/upgoer5/
